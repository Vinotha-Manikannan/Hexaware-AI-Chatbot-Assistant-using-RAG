SECTION 2: LAPTOP & HARDWARE SUPPORT
================================================================

IF YOUR LAPTOP IS NOT WORKING
  Step 1 : Restart the laptop first (resolves 60% of issues)
  Step 2 : If issue persists, raise an IT ticket on portal
  Step 3 : IT team contacts you within 2 hours
  Step 4 : Loaner laptop issued immediately while yours is repaired
  Step 5 : Repair timeline:
           Minor issues    : 24 hours
           Major issues    : 3-5 working days
           Hardware replacement : 7-10 working days
  Step 6 : If repair exceeds 14 days → new laptop issued

LAPTOP SLOW PERFORMANCE
  Self-help steps:
  1. Restart your laptop completely (not sleep/hibernate)
  2. Close unused browser tabs and applications
  3. Run Disk Cleanup: search "Disk Cleanup" in Windows Start
  4. Check for Windows updates: Settings → Windows Update
  5. If still slow after these steps → raise IT ticket

BLUE SCREEN / SYSTEM CRASH
  - Note the error code shown on blue screen
  - Call IT Help Desk immediately: 1800-XXX-XXXX
  - Do NOT restart repeatedly — wait for IT guidance
  - IT will remotely diagnose or collect device

LAPTOP OVERHEATING
  - Place on hard flat surface — not on bed or sofa
  - Clean air vents (can use compressed air)
  - If fan is making noise → raise IT ticket immediately

SECTION 3: VPN SETUP & CONNECTIVITY
================================================================

VPN SOFTWARE : Cisco AnyConnect / Palo Alto GlobalProtect
VPN Server   : vpn.hexaware.com

HOW TO CONNECT TO VPN FROM HOME
  Step 1 : Download VPN software
           Cisco AnyConnect: https://vpn.hexaware.com/download
           Or from IT Help Desk Portal → Downloads
  Step 2 : Install the software
  Step 3 : Open application → enter server: vpn.hexaware.com
  Step 4 : Enter your Windows username and password
  Step 5 : Complete MFA (Microsoft Authenticator approval)
  Step 6 : Connected — green icon shows in taskbar

VPN TROUBLESHOOTING
  Cannot connect to VPN:
  → Check your internet connection first
  → Ensure VPN software is latest version
  → Try disconnecting and reconnecting
  → Check MFA app is working on your phone
  → If still failing: call IT Help Desk 1800-XXX-XXXX

VPN DISCONNECTS FREQUENTLY:
  → Check home internet stability
  → Switch from WiFi to wired LAN if possible
  → Restart VPN software
  → Raise IT ticket if issue continues

SECTION 5: PASSWORD MANAGEMENT
================================================================

PASSWORD POLICY
  Minimum length      : 12 characters
  Must include        : Uppercase, lowercase, number, special character
  Password expiry     : Every 90 days
  Cannot reuse        : Last 10 passwords

HOW TO RESET YOUR PASSWORD
  Self-Service (recommended — available 24/7):
  1. Go to: https://passwordreset.hexaware.com
  2. Enter your Employee ID
  3. Verify via Microsoft Authenticator app
  4. Set new password

  Through IT Help Desk:
  1. Call: 1800-XXX-XXXX (identity verification required)
  2. IT resets and provides temporary password
  3. You must change password on first login

ACCOUNT LOCKED
  - Happens after 5 incorrect password attempts
  - Self-unlock: https://passwordreset.hexaware.com
  - Or call IT Help Desk: 1800-XXX-XXXX

HOW TO SET UP MICROSOFT AUTHENTICATOR (MFA)
  1. Download Microsoft Authenticator from App Store / Play Store
  2. Go to: https://aka.ms/mysecurityinfo
  3. Sign in with company email
  4. Click Add Method → Authenticator App
  5. Scan QR code shown on screen
  6. Approve test notification
  7. MFA setup complete

SECTION 6: EMAIL & MICROSOFT 365

Company Email Format: firstname.lastname@hexaware.com

HOW TO SET UP OUTLOOK ON NEW DEVICE
  1. Open Outlook
  2. Enter your company email address
  3. Sign in with company credentials
  4. Approve MFA on phone
  5. Outlook configures automatically

OUTLOOK NOT SYNCING / EMAILS DELAYED
  Self-help:
  1. Check internet/VPN connection
  2. Click Send/Receive All Folders (F9)
  3. Restart Outlook
  4. Quick Repair: Control Panel → Programs → Microsoft 365 → Quick Repair
  5. If issue persists → raise IT ticket

MICROSOFT TEAMS NOT WORKING
  1. Sign out and sign back in
  2. Clear Teams cache: close Teams → delete C:\Users\[name]\AppData\Roaming\Microsoft\Teams
  3. Restart Teams
  4. Reinstall if needed (IT Help Desk portal)

MAILBOX STORAGE FULL
  - Default mailbox size: 50 GB
  - Archiving enabled after 30 days
  - Request additional storage via IT portal if needed
  - Delete or archive large attachments from Sent/Inbox

SECTION 8: PRINTER SETUP & ISSUES
================================================================

HOW TO CONNECT TO OFFICE PRINTER
  1. Open Settings → Bluetooth & Devices → Printers & Scanners
  2. Click Add Device
  3. Select your floor printer (e.g., PRINTER-FL3-A)
  4. Driver installs automatically
  5. Print test page to confirm

PRINTER NOT WORKING
  Self-help:
  1. Check printer is powered on and has paper
  2. Check printer display for error message
  3. Cancel all print jobs in queue (Settings → Printers → Open Queue → Cancel All)
  4. Restart printer using power button
  5. Reconnect to printer on laptop
  6. If still not working → raise IT ticket (toner/hardware issue)

SECTION 9: DATA BACKUP & RECOVERY
================================================================

  - All company data must be stored on OneDrive or SharePoint
  - Local laptop storage is NOT backed up — do NOT store important files only on C: drive
  - OneDrive auto-syncs all files in real time
  - Deleted files recoverable from OneDrive Recycle Bin for 90 days

HOW TO RECOVER DELETED FILE
  1. Go to: https://hexaware.sharepoint.com
  2. Open OneDrive → Recycle Bin
  3. Find deleted file → Restore
  4. For files deleted more than 90 days ago → raise IT ticket (backup restore)
